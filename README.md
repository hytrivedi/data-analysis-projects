#  Financial Analysis of Indian Manufacturing Firms (2017–2024)

##  Overview
This project analyzes financial data of 101 listed Indian manufacturing companies over the period 2017–2024. The objective was to understand the relationship between inventory management efficiency and firm profitability.

##  Objective
To examine whether inventory turnover has a significant impact on profitability and derive insights for better inventory decision-making.

##  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Excel (Data Cleaning & Validation)

##  Dataset
- Financial data of 101 publicly listed Indian manufacturing firms
- Time period: 2017–2024

##  Analysis Performed
- Data cleaning and preprocessing
- Calculation of financial ratios (Inventory Turnover, Profitability metrics)
- Correlation analysis between key variables
- Data visualization for trend and relationship analysis

##  Key Insights
- Identified a **significant negative correlation** between inventory turnover and profitability
- Firms with excessively high inventory turnover showed reduced profitability in several cases
- Suggests the need for balanced inventory management rather than aggressive turnover strategies

##  Files Included
- `analysis.py` → Python code for data cleaning and analysis

##  Conclusion
The analysis highlights that higher inventory turnover does not always translate into higher profitability. Companies should adopt optimized inventory strategies aligned with demand and operational efficiency.
